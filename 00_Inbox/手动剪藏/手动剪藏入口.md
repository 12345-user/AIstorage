# 手动剪藏入口

把网页、社媒链接、截图说明、课程资料、临时灵感先放在这里。本入口使用 `06_Skills/知识库输入标准与指令识别规范.md`。

```markdown

- 输入时间：YYYY-MM-DD HH:mm
- 来源类型：网页 / 论文 / 课程 / 会议 / 社媒 / 视频 / 文档 / 手动记录 / 其他
- 来源名称：
- 作者 / 机构：
- 原始日期：
- 原始链接：
- 项目归属：P1 / P2 / P3 / P4 / P5 / X / 待判断
- 内容类型：资料
- 目标输出：文章 / 报告 / 视频脚本 / 视频摘要 / 课程 / 周报 / 每日简报 / 暂无
- 处理指令：整理 / 入库 / 选题 / 知识卡 / 脚本 / 文章 / 报告 / 周报 / 简报 / 复盘 / 归档 / 忽略
- 分类动作：P / X / S / A / B / C / D
- 优先级：高 / 中 / 低
- 状态：待处理
- 标签：

## 一句话摘要


## 原始内容


## 推荐去向

- 保存路径：
- 关联笔记：
- 下一步动作：
```

## 可直接写的指令

```text
CMD:整理 | PROJECT:P4 | ACTION:B+A | OUTPUT:知识卡 | PRIORITY:中
```

## 处理规则

Codex 处理后，应把内容移动或复制到：

- `01_Sources`：原始资料。
- `02_Knowledge`：知识卡片。
- `03_Topics`：选题或研究问题。
- `10_Projects`：具体项目资料。
- `20_Shared`：跨项目复用资料。
