# 抓取源配置清单

更新时间：2026-06-08

本清单使用统一识别字段：`CMD`、`PROJECT`、`ACTION`、`OUTPUT`、`TARGET`、`PRIORITY`。抓取到的单条资料使用 `_templates/自动抓取资料模板.md`。

## 抓取源标准格式

```text
来源名称：
来源类型：RSS / GitHub / 论文 / 网页 / 社媒 / 其他
原始链接：
PROJECT：P1 / P2 / P3 / P4 / P5 / X / 待判断
CMD：整理
ACTION：B / S / A / P / X / C / D
OUTPUT：文章 / 报告 / 视频脚本 / 视频摘要 / 周报 / 每日简报 / 暂无
TARGET：00_Inbox/自动抓取
PRIORITY：高 / 中 / 低
抓取频率：
关键词：
授权状态：无需 / 待授权 / 已授权
```
